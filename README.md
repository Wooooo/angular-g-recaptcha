# angular-g-recaptcha
Angular module for google recaptcha

<a name="installation"></a>
## Installation

npm

    npm install angular-g-recaptcha

bower

    bower install angular-g-recaptcha --save
    
<a name="feature"></a>
## Features

- load recaptcha script automatically
- wrap callbacks with $apply
